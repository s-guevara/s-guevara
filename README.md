### Terrorism Activity Around the World and the United States

This visualization project take a look at terrorist activity in the last decade or so around the world and in the United States.

Data used for this project comes from:
- The [Global Terrorism Database](https://www.start.umd.edu/gtd/), for data on global terrorism
- [The Washington Post](https://github.com/wpinvestigative/csis_domestic_terrorism), for data on domestic terrrorism
- The World Bank, for data on country income level classification
