Washington Post

The domestic terrorism data using in this project was collected from CSIS's latest report by the Washington Post and shared via GitHub. The variables used here originated both from CSIS and The Washington Post. For more information on the data visit The Posts's GitHub.
For more analysis on extremism in the US, specifically right-wing extremism, you can visit Andrew Ba Tran's The Washington Post Right-wing Extremism Toplines Flow.

World Bank

World Bank data was collected using the WBGAPI. World Bank data was used to distinguish countries by income level.
